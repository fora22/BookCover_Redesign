# Overview
팽이와 나뚜의 북커버 다시 만들기 프로젝트!!

좋은 책에는 좋은 북커버가 필요하다.

외국책이 국내에서 번역되어 들어올 때, 국내의 감성(?)에 알맞게 북커버 및 북디자인도 변형된다.

초보 개발자 팽이와 초보 디자이너 나뚜는 **이번 기회에 게임을 그만두고 북커버를 리메이크해보려 한다!!**
